# Stargate: Destiny — Una Serie Cinematográfica para Blender

**La continuación de Stargate Universe. 3 temporadas. 39 episodios. 21,000+ líneas de guión.**

---

## ¿De qué trata?

Diez años después de que la Destiny se perdiera en el borde del universo, la Tierra establece una conexión permanente con la nave y descubre que **Eli Wallace sigue vivo** — ha pasado 8 años despierto, solo, reparando la nave, envejeciendo, cambiando.

La Destiny ha cruzado galaxias siguiendo un mapa dejado por las sembradoras. Al final del viaje: el **código fuente cosmológico** — el sistema operativo de la realidad, sembrado al inicio del universo. Pero una civilización más antigua que los Antiguos, **Los Curadores**, lleva eones eliminando a quienes se acercan demasiado a la verdad.

Temporada 1 | *El Despertar* — Eli solo. La lucha de Carter. La conexión.
Temporada 2 | *El Mapa* — Las sembradoras. Los fragmentos del código. "Ella".
Temporada 3 | *El Origen* — La Estructura. El sacrificio. La amenaza que viene.

## Cómo contribuir

Este proyecto es **abierto a la comunidad**. Si eres fan de Stargate, escritor, artista 3D, o simplemente tienes ideas, puedes aportar de muchas maneras:

### 🖊️ Como escritor / guionista

- **Revisa los guiones existentes** — ¿un diálogo no funciona? ¿una escena necesita más tensión? Abre un issue o un PR.
- **Propón episodios de relleno** — la serie tiene espacio para episodios que desarrollen personajes, exploren planetas, o añadan capas al misterio.
- **Escribe variaciones** — ¿y si el final fuera diferente? ¿y si otro personaje se sacrificara? Los "what if" son bienvenidos.

### 🎨 Como artista 3D / Blender

- La serie está diseñada para **producción hiperrealista en Blender**.
- Consulta `series/blender/assets.md` para ver la lista de assets necesarios.
- Modela, texturiza, riggea personajes, naves, sets.
- Cada episodio tiene notas `[BLENDER]` que especifican cámaras, luces, VFX.

### 🔬 Como científico / asesor técnico

- ¿Las matemáticas del código fuente no cuadran? ¿La física del FTL necesita ajustes?
- Abre un issue con tu corrección. El realismo técnico importa.

### 💬 Como fan

- **Feedback narrativo**: ¿un personaje no funciona? ¿un arco se siente forzado?
- **Teorías**: ¿qué crees que significa el post-créditos? ¿quién es la figura del cristal?
- **Correcciones de lore**: ¿algo contradice el canon de Stargate?

## Cómo empezar a contribuir

1. **Haz fork** del repositorio.
2. **Elige un área**: guiones, blender, lore, correcciones.
3. **Lee los archivos relevantes** en `series/`.
4. **Abre un issue** para discutir tu propuesta antes de enviar un PR.
5. **Envía tu PR** con una descripción clara de los cambios.

### Guía rápida de estilo para guiones

Los guiones usan formato profesional:

```
**INT. DESTINY - SALA DE COMPUERTAS - DÍA**

La compuerta se activa. El vórtice azul estabiliza.

**ELI** (susurrando a Wilson)
Aquí vamos.

**CARTER**
(cruzando el portal)
Eli. ¿Eres tú?
```

- **INT/EXT. LOCACIÓN - DÍA/NOCHE** para encabezados
- Acción en presente, descriptiva pero concisa
- **PERSONAJE:** para diálogo
- *(paréntesis)* para tono o acción durante el diálogo
- `[BLENDER]` para notas de producción visual
