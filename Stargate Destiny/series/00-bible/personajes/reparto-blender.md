## Configuración de personajes para animación

- **Rigging**: Rig humano completo para todos los personajes (Face Rig + Body Rig)
- **Face Morphs**: Expression set estándar (alegría, tristeza, enojo, sorpresa, miedo, desprecio)
- **Blend Shapes**: Cada personaje necesita blend shapes para diálogo visema (inglés)
- **Ciclos de caminata**: Marcha militar (Chen, Hawk), marcha científica (Eli, McKay), paso firme (Teal'c)
- **Texturas**: 4K para personajes principales, 2K para secundarios. Piel con textura procedural de poros/subdermal.
